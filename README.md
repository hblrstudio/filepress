# FilePress

Compress images and PDFs to any target size. Desktop app for macOS and Windows.

**→ [filepressapp.vercel.app](https://filepressapp.vercel.app)**

---

## What it does

- Type a target size ("500 KB", "2 MB") — FilePress hits it automatically
- Or use the quality slider for manual control
- Platform presets: Instagram, WhatsApp, Email, Twitter/X, LinkedIn, PDF Web
- Supports JPEG, PNG, WEBP, HEIC, PDF
- 100% offline — your files never leave your machine

## Stack

- Python 3.13 + CustomTkinter
- Pillow (image processing), pikepdf (PDF compression)
- PyInstaller (standalone .app / .exe bundle)
- Next.js landing page on Vercel
